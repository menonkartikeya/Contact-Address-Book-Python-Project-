If you want to just run and use this project:
Download the "executable" of this project, i.e. download and run PrefixInfixPostfix.exe on your PC.


This app changes entire background color everytime something is calculated. 
(Added this feature just to make the app a little more fun and a little less boring.)


For developers:

In this folder: Open 'PrefixInfixPostfix.py' and run.
All the functionalities are easy to understand. 

Only one icon is used in this project. 
Credits for icon: "Icon made by DinosoftLabs from www.flaticon.com"

The file "functions.py" is used here for defining the functions of (and accessing the) Converting one expression to other two.
List of all functions in it:
    prefixToInfix
    prefixToPostfix
    infixToPrefix
    infixToPostfix
    postfixToPrefix
    postfixToInfix
These functions are called accordingly, when "Calculate" button is clicked.
