In the folder "Contacts - menonkartikeya": Open 'index.py' and run.
Have a play around with it. All the functionalities are easy to understand. 

The subfolder "Icons" contains all the necessary icons used in this project. 
Credits :https://www.flaticon.com/

Contacts_Data.csv is a sample csv file with some contacts stored already. 
It is to try and experience the "import contacts" functionality.

One can also export contacts from their database to a .csv file using "export contacts" option.

The file "info.py" is used here for defining the functions of (and accessing the) "info" option from menubar.
